Curriculum Overview

Our comprehensive curriculum is designed to make you job-ready

Weeks 1-4: Foundations

  • HTML5, CSS3, JavaScript ES6+
  • Git, GitHub, and development workflow
  • Responsive design and accessibility
  • Building your first web applications

Weeks 5-8: Frontend Development

  • React.js and component-based architecture
  • State management with Redux
  • API integration and asynchronous JavaScript
  • Testing and debugging

Weeks 9-12: Backend Development

  • Node.js and Express.js
  • Database design with MongoDB and SQL
  • RESTful API development
  • Authentication and security
  • Deployment and DevOps basics

Weeks 1-4: Core Frontend Technologies

  • Advanced HTML5 and semantic markup
  • CSS3, Flexbox, Grid, and animations
  • JavaScript fundamentals and DOM manipulation
  • Responsive design principles

Weeks 5-7: Modern Frameworks

  • React.js fundamentals and hooks
  • Component architecture and props
  • State management and context API
  • React Router for SPA development

Weeks 8-10: Advanced Frontend

  • TypeScript for scalable applications
  • Testing with Jest and React Testing Library
  • Performance optimization techniques
  • Accessibility (a11y) best practices
  • Build tools and deployment

Weeks 1-4: Backend Fundamentals

  • Node.js runtime and module system
  • Express.js framework and middleware
  • RESTful API design principles
  • Authentication with JWT and OAuth

Weeks 5-7: Databases and Data Modeling

  • SQL databases (PostgreSQL) and ORMs
  • NoSQL databases (MongoDB) and ODMs
  • Database design and normalization
  • Query optimization and indexing

Weeks 8-10: Advanced Backend

  • Microservices architecture
  • Message queues and background jobs
  • Caching strategies with Redis
  • API security and best practices
  • Containerization with Docker

Weeks 1-5: Data Fundamentals

  • Python for data science (NumPy, Pandas)
  • Data cleaning and preprocessing
  • Exploratory data analysis and visualization
  • Statistical analysis and hypothesis testing

Weeks 6-10: Machine Learning

  • Supervised learning algorithms
  • Unsupervised learning and clustering
  • Model evaluation and validation
  • Feature engineering and selection

Weeks 11-14: Advanced Topics

  • Deep learning with TensorFlow/Keras
  • Natural language processing
  • Time series analysis and forecasting
  • Big data tools and cloud platforms
  • Deploying machine learning models
